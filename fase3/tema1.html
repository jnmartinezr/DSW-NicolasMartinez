Sintaxis y Selectores CSS

La estructura de CSS se compone de 3 elemento básicos:

- Selector
- Propiedad
- Valor

De la forma:
Selector {propiedad:valor;}
Donde el lector filtra la etiqueta HTML que corresponde al estilo a usar, asimismo; la propiedad, equivale a la característica o sección en la cual se aplica el cambio, y el valor es el cambio que se hará, de manera que, hay un componente que elige el estilo a usar, un componente que indica la sección y otro que indica el cambio especifico que se ejecutará. Como regla destacada, se pueden agrupar ediciones de estilo con, punto y coma (;) para abreviar la redacción.
