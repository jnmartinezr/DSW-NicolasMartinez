Interacción Humano Ordenador


La interacción persona ordenar, o Human-Computer Interaction (HCI), se define como la gestión completa de sistemas informáticos, con componentes interactivos, diseñados para el humano, estudiado desde los puntos mas trascendentales para el ser humano. En una gran cantidad de países por todo el mundo, se estudia al respecto, pero en Estados Unidos, se encuentra la mayor concentración de aportes a este tema. Ahora bien, teniendo en cuenta que el tema anterior se enfoca en CSS, es conveniente continuar la misma línea temática, agregando a lo ya mencionado que, la interacción que se busca se puede entender como una comunicación entre el humano y el sistema con el cual interactúa, por lo tanto entre más eficiente sea esta comunicación más agradable resulta al usuario y por ende mas practico, y esta última característica esta incluida dentro de la “interacción natural” que, corresponde a la manera intuitiva en la que se presenta un sitio web al usuario para que le resulte más cómodo reconocer y trabajar con los componentes de los cuales disponga el sitio. Entonces, el enfoque radica en generar contenido, cuya accesibilidad sea evidente y fluida no solo para un tipo de personas en especifico sino para cualquiera que acceda a un sitio Web de esta clase.
