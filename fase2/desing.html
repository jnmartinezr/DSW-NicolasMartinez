Sintaxis y Selectores CSS

La estructura de CSS se compone de 3 elemento básicos:

- Selector
- Propiedad
- Valor

De la forma:
Selector {propiedad:valor;}
Donde el lector filtra la etiqueta HTML que corresponde al estilo a usar,
asimismo; la propiedad, equivale a la característica o sección en la cual
se aplica el cambio, y el valor es el cambio que se hará, de manera que,
hay un componente que elige el estilo a usar, un componente que indica la
sección y otro que indica el cambio especifico que se ejecutará.
Como regla destacada, se pueden agrupar ediciones de estilo con, punto y coma (;)
para abreviar la redacción.

Interacción Humano Ordenador


La interacción persona ordenar, o Human-Computer Interaction (HCI), se
define como la gestión completa de sistemas informáticos, con componentes
interactivos, diseñados para el humanos estudiado desde los puntos mas
trascendentales para el ser humano.
En una gran cantidad de países por todo el mundo, se estudia al respecto,
pero en Estados Unidos, se encuentra la mayor concentración de aportes a este tema.


Granollers i Saltiveri, T. (2012).
Diseño de sistemas interactivos centrados en el usuario.
Barcelona, Spain: Editorial UOC. Recuperado de
